# Maze Solver

A visual maze generator and solver built with Python and Tkinter GUI.

This is the starter code used in Boot.dev's [Build a Maze Solver in Python](https://www.boot.dev/courses/build-maze-solver-python) course.

---

## Requirements

- Python 3
- Tkinter (usually preinstalled with Python)

---

## Installation

1. Clone the repository.

2. Install Tkinter manually if needed:
```bash
sudo apt-get install python3-tk
```

---

## Usage

### Run maze solver

```bash
python3 main.py
```
